# Spacer

Use a `Spacer` to create flexible space inside a stack.

```json
[
    {
        "type": "HStack",
        "children": [
            {
                "type": "Text",
                "title": "Leading"
            },
            {
                "type": "Spacer"
            },
            {
                "type": "Text",
                "title": "Trailing"
            }
        ]
    }
]
```

## Parameters

| Parameter | Type | Description |
| --- | --- | --- |
| `modifiers` | Object | Visual and behavioral modifiers. |
