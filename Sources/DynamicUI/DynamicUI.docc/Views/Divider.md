# Divider Example

This example demonstrates how to define a `Divider` using DynamicUI's JSON schema.  

```json
[
    {
       "type": "Divider"
    }
]
```

### Parameters

| Parameter | Type        | Description                       |
| --------- | ----------- | --------------------------------- |
| title     | String      | The title of the divider.        |
| modifiers | Object      | The visual modifiers for the divider. |
